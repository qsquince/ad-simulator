# 广告动态出价模拟器 – 策略产品原型

## 背景与痛点
中小广告主面临：预算有限、流量分时波动、不知如何动态出价。  
本产品帮助投放人员**对比不同出价策略的 ROI 与消耗趋势**，降低试错成本。

## 核心功能
- 4 种出价策略：匀速 / 激进 / 保守 / 动态最优控制（基于剩余预算与竞争）
- 24 小时 CTR 乘数调节，模拟真实流量峰谷
- 7 天 ROI & 消耗趋势预测
- 预算撞线预警 + 策略建议

## 在线体验
 https://github.com/qsquince/ad-simulator

## 产品设计思路（重点）
- **动态策略算法**：结合预算进度（β）和竞争对手出价（α），实时调整出价，避免预算过早耗尽。
- **交互设计**：参数滑杆实时联动，导出图片报告，降低使用门槛。
- **适用场景**：电商拉新、App 买量、本地生活投放。
